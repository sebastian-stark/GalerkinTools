\doxysubsection{Detailed Description}
\subsubsection*{template$<$unsigned int spacedim$>$\newline
class Point\+Constraint$<$ spacedim, spacedim $>$}

Class defining a Dirichlet type condition for a domain related field at a single point.

The Dirichlet type condition has the form $u^\Omega_\epsilon(\boldsymbol{X}) = b^\Omega_\epsilon + c^\Omega_\epsilon C_\iota$, where $b^\Omega_\epsilon$ and $c^\Omega_\epsilon$ are prescribed values. The \mbox{\hyperlink{class_point_constraint_3_01spacedim_00_01spacedim_01_4}{Point\+Constraint$<$spacedim,spacedim$>$}} class inherits from \textbf{ Subscriptor} in order to be able to check that \mbox{\hyperlink{class_point_constraint_3_01spacedim_00_01spacedim_01_4}{Point\+Constraint$<$spacedim,spacedim$>$}} objects are only destroyed when they are not needed anymore by other objects.
