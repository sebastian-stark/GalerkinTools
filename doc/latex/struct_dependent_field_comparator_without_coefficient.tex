A custom comparator for dependent field terms. This is used to allow for std\+::sets of dependent fields in \mbox{\hyperlink{class_dependent_field}{Dependent\+Field}} and \mbox{\hyperlink{class_dependent_field_3_01spacedim_00_01spacedim_01_4}{Dependent\+Field$<$spacedim,spacedim$>$}}. A comparison operator, which allows to use \mbox{\hyperlink{class_dependent_field_term}{Dependent\+Field\+Term}} in {\ttfamily std\+::map} and {\ttfamily std\+::set}. The comparison operator is based on a lexicographic ordering according to the members \mbox{\hyperlink{class_dependent_field_term_a89d1c3fea36e6fe105232097a321e095}{Dependent\+Field\+Term\+::independent\+\_\+field}}, \mbox{\hyperlink{class_dependent_field_term_ac6f3ac40d4ee2c8b9f9bbdfa34079b74}{Dependent\+Field\+Term\+::component}}, \mbox{\hyperlink{class_dependent_field_term_af09c5452c3e8e71e9ee99db304b90135}{Dependent\+Field\+Term\+::derivatives}}. This means that the member \mbox{\hyperlink{class_dependent_field_term_a59a9183a32ac55fb728f3797b68a9f8f}{Dependent\+Field\+Term\+::coefficient}} does not factor into the comparison, which effectively means that two \mbox{\hyperlink{class_dependent_field_term}{Dependent\+Field\+Term}} objects are considered equal if they are associated with the same \mbox{\hyperlink{class_dependent_field_term_a89d1c3fea36e6fe105232097a321e095}{Dependent\+Field\+Term\+::independent\+\_\+field}}, \mbox{\hyperlink{class_dependent_field_term_ac6f3ac40d4ee2c8b9f9bbdfa34079b74}{Dependent\+Field\+Term\+::component}}, \mbox{\hyperlink{class_dependent_field_term_af09c5452c3e8e71e9ee99db304b90135}{Dependent\+Field\+Term\+::derivatives}} (the same \mbox{\hyperlink{class_dependent_field_term_a89d1c3fea36e6fe105232097a321e095}{Dependent\+Field\+Term\+::independent\+\_\+field}} here really means \char`\"{}exactly the same object\char`\"{}).
