\subsection{Detailed Description}
This class is intended to describe a renumbering of dof indices

The class inherits from {\bf Subscriptor} in order to be able to check that \hyperlink{class_do_f_renumbering}{Do\+F\+Renumbering} objects are only destroyed when they are not needed anymore by other objects.

The default implementation does no renumbering.
