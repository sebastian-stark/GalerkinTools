\doxysubsection{Detailed Description}
\subsubsection*{template$<$unsigned int dim, unsigned int spacedim = dim$>$\newline
class parallel\+::\+Triangulation$<$ dim, spacedim $>$}

This class derives from \textbf{ parallel\+::distributed\+::\+Triangulation} and provides an interface triangulation appropriate for use within the parallel\+::distributed\+::\+Triangulation\+System. The main additional functionality of this class over \textbf{ parallel\+::distributed\+::\+Triangulation} is that the subdomain ids can be assigned manually. The triangulation can only work properly, if the partitioning is done consistently.
