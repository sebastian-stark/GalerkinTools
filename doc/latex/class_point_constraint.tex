\doxysubsection{Detailed Description}
\subsubsection*{template$<$unsigned int dim, unsigned int spacedim$>$\newline
class Point\+Constraint$<$ dim, spacedim $>$}

Class defining a Dirichlet type condition for an interface related field at a single point.

The Dirichlet type condition has the form $u^\Sigma_\eta(\boldsymbol{X}) = b^\Sigma_\eta$, where $b^\Sigma_\eta$ is a prescribed value. The \mbox{\hyperlink{class_point_constraint}{Point\+Constraint}} class inherits from \textbf{ Subscriptor} in order to be able to check that \mbox{\hyperlink{class_point_constraint}{Point\+Constraint}} objects are only destroyed when they are not needed anymore by other objects.


\begin{DoxyTemplParams}{Template Parameters}
{\em dim} & The dimension of the object on which the independent field is defined. Note that also dim=0 is admissible in order to constrain an independent scalar. In this case, however, \mbox{\hyperlink{class_point_constraint_ac021367793862b52580cc292bb7b0ea7}{Point\+Constraint\+::X}} is ignored\\
\hline
{\em spacedim} & The spatial dimension of the problem \\
\hline
\end{DoxyTemplParams}
