\doxysubsection{Detailed Description}
\subsubsection*{template$<$unsigned int dim, unsigned int spacedim$>$\newline
class Function\+Cell$<$ dim, spacedim $>$}

Special \textbf{ Function} class allowing to pass the cell information when asking for the initial value of an unknown field at a finite element support point. Objects of this type can be given to the constructor of an \mbox{\hyperlink{class_independent_field}{Independent\+Field}}; and when the initial solution vector is set up with a call to \mbox{\hyperlink{class_assembly_helper_a4043c993c8902ad5d3045a13ed42e6f8}{Assembly\+Helper\+::get\+\_\+initial\+\_\+fields\+\_\+vector()}}, \mbox{\hyperlink{class_function_cell_a3ef39628d9141076b3418fe2bb529dfe}{Function\+Cell\+::cell}} will be appropriately set before each call to the \textbf{ Function\+::value()}, so that one can find out through \mbox{\hyperlink{class_function_cell_ab3f22f3fa0cb7c3eec847f883fb7fecd}{Function\+Cell\+::get\+\_\+cell()}} on which cell the support point lies, for which \textbf{ Function\+::value()} currently has to provide with initial values.
