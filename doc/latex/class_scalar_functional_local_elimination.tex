\doxysubsection{Detailed Description}
\subsubsection*{template$<$unsigned int dim, unsigned int spacedim$>$\newline
class Scalar\+Functional\+Local\+Elimination$<$ dim, spacedim $>$}

An interface related scalar functional with functionality to eliminate local dependent fields. This scalar functional uses a Newton-\/\+Raphson algorithm in order to determine new values of the dependent fields to be eliminated such that the gradient of the scalar functional w.\+r.\+t. these dependent fields is zero. The scalar functional may be constructed from several other scalar functionals (as a sum of these).
