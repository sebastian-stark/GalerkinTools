Detailed Description

template<unsigned int spacedim>
class PointConstraint< spacedim, spacedim >

Class defining a Dirichlet type condition for a domain related field at a single point.

The Dirichlet type condition has the form , where and are prescribed values. The PointConstraint<spacedim,spacedim> class inherits from Subscriptor in order to be able to check that PointConstraint<spacedim,spacedim> objects are only destroyed when they are not needed anymore by other objects.

const SmartPointer<const Function<spacedim> > PointConstraint< spacedim, spacedim >::constraint_inhomogeneity

A Function (or, rather, an instance of a derived class) specifying the constraint inhomogeneity . The position variable of the function object is ignored. However, the Function::set_time() or PointConstraint<spacedim, spacedim>::set_time() functionality can be used to change the constraint inhomogeneity with time. The Function must have a single component and implement the method Function::value(). If a nullptr is stored here, the constraint will be assumed homogeneous.

template<unsigned int spacedim>
const bool PointConstraint< spacedim, spacedim >::ignore_point = false

If this is true, the point PointConstraint::X is ignored and an arbitrary dof index of the independent field is constrained. This is useful to fix the constant of a field, which is discretized by a finite element without support points.
