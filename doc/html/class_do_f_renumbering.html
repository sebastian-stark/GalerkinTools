Detailed Description

This class is intended to describe a renumbering of dof indices

The class inherits from Subscriptor in order to be able to check that DoFRenumbering objects are only destroyed when they are not needed anymore by other objects.

The default implementation does no renumbering.

Constructor & Destructor Documentation

virtual DoFRenumbering::~DoFRenumbering ( )

The destructor of DoFRenumbering essentially checks before destruction that the DoFRenumbering object is not used by other objects. If this is the case, the program will be aborted.
