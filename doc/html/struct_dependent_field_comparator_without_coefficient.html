Detailed Description

template<unsigned int dim, unsigned int spacedim>
struct DependentFieldComparatorWithoutCoefficient< dim, spacedim >

A custom comparator for dependent field terms. This is used to allow for std::sets of dependent fields in DependentField and DependentField<spacedim,spacedim>. Coefficients of the dependent field terms are not included in the comparison.

A comparison operator, which allows to use DependentFieldTerm in std::map and std::set. The comparison operator is based on a lexicographic ordering according to the members DependentFieldTerm::independent_field, DependentFieldTerm::component, DependentFieldTerm::derivatives. This means that the member DependentFieldTerm::coefficient does not factor into the comparison, which effectively means that two DependentFieldTerm objects are considered equal if they are associated with the same DependentFieldTerm::independent_field, DependentFieldTerm::component, DependentFieldTerm::derivatives (the same DependentFieldTerm::independent_field here really means "exactly the same object").
