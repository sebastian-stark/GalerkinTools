Detailed Description

This renumbering scheme is based on applying offsets to index ranges. Presently no measures are taken to ensure that the overall index set is consistent!

◆ dof_offsets

std::vector<std::tuple<const unsigned int, const unsigned int, const int> > DoFRenumberingOffset::dof_offsets
private

This data structure defines a fixed shift of dof numbers from a certain interval [dof_start_i, dof_end_i] to the interval [dof_start_i + offset_i, dof_end_i + offset_i]. The tuples (dof_start_i, dof_end_i, offset_i) define this shift. Multiple intervals can be added and the order of adding the intervals influences how the intervals are searched during conversion of dof indices. In particular, if a certain dof index is to be converted, it is initially tried to find it in the interval added first, if it is not found there, the second interval is tried, and so forth.
