Detailed Description

A SolverWrapper for the MA57 solver.

Note
MA57 has its own license, independent of that of GalerkinTools. If you want to use MA57 you have to accept that license and obtain your own copy of MA57.

Code partially copied over from deal.II

◆ analyze

unsigned int BlockSolverWrapperMA57::analyze = 0

Key indicating when to analyze the matrix structure:
0 - before each factorization (default)
1 - only before the next factorization (afterwards, BlockSolverWrapperMA57::analyze will be set to 2)
>=2 - do not recompute

Warning
The user must ensure that BlockSolverWrapperMA57::analyze=0 or BlockSolverWrapperMA57::analyze=1 whenever the sparsity pattern of the matrix has changed (or during the first call). Currently, no internal checking is performed.
The scaling of the matrix is only recomputed if BlockSolverWrapperMA57::analyze <2. Otherwise, the most recently computed scaling will be used.

◆ ordering_method

unsigned int BlockSolverWrapperMA57::ordering_method = 5

ordering method (key corresponding to ICNTL(6) of MA57):
0: AMD ordering using MC47 (without the detection of dense rows)
2: AMD ordering using MC47
3: Minimum degree ordering using MA27
4: nested dissection (METIS)
5: Automatic choice >=6: currently equivalent to 5
