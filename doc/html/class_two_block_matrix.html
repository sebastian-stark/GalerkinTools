Detailed Description

template<class MatrixType>
class TwoBlockMatrix< MatrixType >

This class implements a block matrix of the form

The main difference to the deal.II block matrices is that internally is stored instead of the transpose of it. This is useful for parallel applications where finite element related dofs are coupled to "global" unknowns which are not related to any mesh.
