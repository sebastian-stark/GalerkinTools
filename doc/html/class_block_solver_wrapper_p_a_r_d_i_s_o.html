Detailed Description

A SolverWrapper for the PARDISO solver.

Note
PARDISO has its own license, independent of that of GalerkinTools. If you want to use PARDISO you have to accept that license.

Code partially copied over from deal.II

unsigned int BlockSolverWrapperPARDISO::analyze = 0

Key indicating when to analyze the matrix structure:
0 - before each factorization (default)
1 - only before the next factorization (afterwards, BlockSolverWrapperPARDISO::analyze will be set to 2)
>=2 - do not recompute

Warning
The user must ensure that BlockSolverWrapperPARDISO::analyze=0 or BlockSolverWrapperPARDISO::analyze=1 whenever the sparsity pattern of the matrix has changed (or during the first call). Currently, no internal checking is performed.
The scaling of the matrix is only recomputed if BlockSolverWrapperPARDISO::analyze <2. Otherwise, the most recently computed scaling will be used.
